🐉 Dragon AR Animation

Unity Augmented Reality Project

📌 Overview

This project is an Augmented Reality (AR) application developed using Unity. It detects a reference image and spawns a 3D dragon model that can be controlled and animated using user input.

🎯 Features
Image-based AR tracking
3D dragon spawning
Joystick-based movement
Directional rotation
Flying animation control
📂 Files in Repository

This repository contains:

ARTrackedImageSpawner.cs → Handles AR image detection and spawning
CharizardController.cs → Controls movement and animation of the dragon
Project Output Image (sample result of AR scene)
⚙️ How It Works
1. Image Detection
Uses Unity AR Foundation
Detects a predefined reference image
2. Object Spawning
Instantiates the dragon prefab on the detected image
3. Movement Control
Controlled using a joystick
Moves along X and Z axis
4. Animation
Flying animation plays when moving
Stops when idle
🛠️ Technologies Used
Unity Engine
AR Foundation
C#
🚀 Usage
Open project in Unity
Attach scripts to appropriate objects
Add reference image in XR Image Library
Run on AR-supported mobile device
